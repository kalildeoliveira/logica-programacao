Aula 3 - Operações lógicas e aritméticas

    Para trabalhar com os dados guardados nas variáveis, podemos fazer diversas coisas como comparações, escolhas, repetições e outros processos. Tal coisa é possível a partir de operadores.

    Pense no tempo do colégio. Operador era aquele "+" entre dois números que se usava para dizer que estava somando. Lembra? Só que neste caso muda um pouco porque depois de somar você não vai usar mais um "=" para mostrar o resultado, mas guardar em uma variável e exibí-la com o comando escreva ()

    Enfim, tal como na vida, em um programa de computador os operadores aritméticos são usados para somar, subtrair, multiplicar e dividir. E neste último caso, é possível usar operadores tanto para ver o resultado quanto o resto da divisão no caso de números inteiros. Veja o exemplo:

      algoritmo resto
      var numero, divisor : inteiro
      inicio
        leia (numero, divisor)
        numero ← numero mod divisor
        escreva (numero)
      fim.

    Neste caso as variáveis numero, divisor receberam um valor digitado pelo usuário. Digamos que sejam, respectivamente, os números 10 e 2. O resto da divisão será zero e será guardada na variável numero usando o comando mod. Escrevemos por cima do valor 10 que estava guardado, ou seja, um novo valor é sobrescrito na variável numero e exibido em seguida com o comando escreva () na penúltima linha do programa.

    Como seria este código em C? Vejamos:

      #include <stdio.h>

      int main () {
        int numero, divisor;
        scanf ("%d %d", &numero, &divisor);
        numero = numero % divisor;
        printf ("%d", numero);
      }

    O Javascript roda o mesmo programa da seguinte forma:

      var numero = parseInt(prompt ());
      var divisor = parseInt(prompt ());
      numero = numero % divisor;
      console.log (numero);


    Neste código você pode perceber que o comando ou função para forçar o usuário a atribuir um valor à variável via teclado, prompt (), roda dentro do comando parseInt () pois o prompt () recebe apenas string, texto, mesmo que seja digitado número e o parseInt () faz a conversão para o tipo inteiro.

    É por isso que, em Javascript, quando alguém quiser somar duas strings que guardam o valor "2", o operador faz uma concatenação, ou seja, 2 + 2 = 22. Para o resultado ser 4, tem que converter inteiro ou real. Interessante também que, escrevendo parseInt(prompt()); economizamos uma linha de código pois seria equivalente a escrever var numero = prompt(); parseInt(numero);. Percebe?

    As operações lógicas envolvem basicamente "e", "ou" e "não". Com uma estrutura de decisão, podemos usar bastante estas operações. Por exemplo, um aluno não pode ser aprovado se ele tiver média sem frequência. Conectando as duas variáveis com o operador lógico & resolvemos este problema.
