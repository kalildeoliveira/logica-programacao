Aula 2 - Atribuir valores às variáveis

    Uma forma comum de iniciar um programa é declarando e iniciando as variáveis. Trata-se do processo de entrada de dados. Em seguida, o código pode prever as regras ou processos a partir dos dados informados e, finalmente, as saídas para o usuário.

    Em pseudocódigo uma declaração com atribuição fica assim:

      algoritmo exemplo
      var nomeUsuario : caracter
      inicio
        nomeUsuario ← João
      fim.

    Neste caso a variável nomeUsuario recebeu um dado de tipo string ou cadeia de caracteres diretamente no código. Mas poderíamos dar a entrada deste dado pelo teclado, ou seja, a partir do comando leia () forçamos o usuário a digitar seu nome. Fica assim:

      algoritmo exemplo
      var nomeUsuario : caracter
      inicio
        leia (nomeUsuario)
      fim.

    Num caminho parecido ao pseudocódigo, a linguagem C exige, porém, o uso dos comandos em inglês e aspectos de sintaxe, como o uso do ponto e vírgula ao final de uma instrução, mas a lógica é a mesma para o que fizemos acima, ou seja, declarar uma variável e guardar um dado informado pelo usuário via teclado. O comando leia () escreve-se scanf (). Veja como fica o mesmo código:

      #include < stdio.h >

      int main () {
        char nomeUsuario;
        scanf ("%s", &nomeUsuario);
      }

    Algo interessante no caso de Javascript é que o programador define o tipo de variável no momento que atribui o seu valor, sem precisar declarar antes, como vimos nas linguagens acima. Uma forma de escrever aquele programinha que guarda o nome de um usuário em uma variável, portanto, vai exige apenas uma linha com o mesmo resultado prático:

      var nomeUsuario = prompt ();

    Como se vê acima, prompt () é o comando que vai exigir do usuário digitar seu nome, que será guardado na variável nomeUsuario e imediatamente identificada pelo computador como string, pois seu conteúdo é um conjunto de caracteres, um texto.
