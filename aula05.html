Aula 5 - Estruturas de repetição

    Uma pessoa bem atenta deve ter percebido na aula anterior que já possui o suficiente para resolver praticamente qualquer problema com programação, afinal já sabe fazer um sistema que recebe uma entrada, faz um processamento e devolve algo ao usuário. O que vem agora, então, é para agregar, aprimorar, aperfeiçoar os códigos.

    Tem a ver principalmente em tornar a vida mais fácil. Afinal, pense o tamanho do código se for preciso criar decisões de aprovação ou reprovação para uma escola com milhares de alunos! Quantos "se...então..." seriam utilizados! Quantas linhas de código! Quanto trabalho!

    As estruturas de repetição são, potanto, a salvação. Graças a elas, podemos repetir uma condição até que o usuário esteja totalmente satisfeito ou todas as possibilidades tenham sido atendidas. E o melhor: sem precisar repetir o código para cada condição. Basta dar o loop, o giro, para tudo se repetir.

    Veja este código:

      algoritmo mediaCom5Notas
      var contador : inteiro
      var nota [5] : real
      nota[0]←0
      inicio
        para contador←1 ate 5 faça
          escreva ("Nota:")
          leia (nota[contador])
          nota[0]←nota[0]+nota[contador]
        fim para
        nota[0]←nota[0]/5
        se (nota[0] >=6) entao
          escreva ("Aprovado")
        senao
          escreva ("Reprovado")
          fim se
        fim se
      fim

    Perceba que colocamos as notas dentro do array nota[5], do tipo real e de seis posições, sendo os índices 1 a 5 para receber as cinco notas e a posição 0 para acumular os valores digitados.

    Obviamente que esta solução serve apenas quando o desenvolvedor sabe que são cinco notas a serem digitadas. E se ele quiser deixar isso para o usuário? Neste caso, basta criar uma variável para determinar o tamanho do array e a quantidade de voltas na instrução "para... faça" ou como condição para parar os giros do comando "enquanto... faça" ou "faça... enquanto".

    Vejamos o código de um "enquanto... faça":

      algoritmo mediaComVariasNotas
      var contador, qtdNotas : inteiro
      var nota [ ] : real
      contador ← 0
      inicio
        escreva ("Quantidade de notas: ")
        leia (qtdNotas)
        enquanto (qtdNotas != contador) faça
          contador++
          escreva ("Nota:")
          leia (nota[contador])
          nota[0]←nota[0]+nota[contador]
        fim enquanto
        nota[0]←nota[0]/qtdNotas
        se (nota[0] >=6) entao
          escreva ("Aprovado")
        senao
          escreva ("Reprovado")
          fim se
        fim se
      fim
