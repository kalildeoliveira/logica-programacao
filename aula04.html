Aula 4 - Controle de decisão

    Em C ou Javascript usa-se && como operador relacional. Imagine como isso pode ajudar no caso de um programa para gerenciar uma escola na qual seus alunos não podem ser aprovados com média sem frequência ou com frequência sem média. Conectando as duas variáveis com o operador lógico "e" resolvemos este problema. É o operador lógico que só vai permitir o fluxo se as duas condições forem atendidas ou verdadeiras.

    Há também algumas situações nas quais apenas uma condição já tá valendo. É o caso de medir a febre de duas pessoas que precisam viajar juntas. Se uma ou as duas pessoas estiverem doentes, a viagem não acontece. O operador de disjunção "ou", que em C e em Javascript se escreve repetindo o símbolo de barra vertical | | , dá sequência ao código bastando que uma condição seja verdadeira.

    Veja como fica este raciocínio quando usamos "ou" associado ao operador de negação "não" e um controle de decisão simples com "se... senão...":

      algoritmo medidorDeFebre
      var pessoa1, pessoa2 : lógico
      inicio
        se não(pessoa1 ou pessoa2) então
          escreva ("Viagem acontece")
          senao
            escreva ("Viagem não acontece")
          fim se
        fim se
      fim
    Considerando que o valor da febre seja Verdadeiro, note que basta uma pessoa ter febre para a viagem não acontecer ou ainda é preciso que as duas estejam com saúde para a viagem acontecer.

    Seria o mesmo que escrever o código assim:

      algoritmo aprovacao
      var media, frequencia : lógico
      inicio
        se (media e frequencia) então
          escreva ("Aprovado")
          senao
            escreva ("Reprovado")
          fim se
        fim se
      fim
    A solução em Javascript fica até bem parecida como o que foi escrito no pseudocódigo:

      var media = frequencia = true;
      if (media && frequencia) {
        console.log ("Aprovado");
      } else {
        console.log ("Reprovado");
      }

    Acima, da direita para a esquerda, o valor lógico true é atribuído à variável frequencia e depois media de modo que as duas passam a ter o mesmo valor. Para ter resultado semelhante, no caso da linguagem C, ao invés de conferir se é verdadeiro ou falso, guarda-se o valor 0 ou 1, como neste programa:

      #include <stdio.h>

      int main () {
        int media = 1, frequencia = 1;
        if ((media ==1)&&(frequencia==1)) {
          printf ("Aprovado");
        } else {
          printf ("Reprovado");
        }
      }
